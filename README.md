# Chiller Peças — site estático pronto para GitHub Pages

Este repositório contém uma landing page institucional moderna para a Chiller Peças.

## Arquivos

- `index.html`: estrutura da página
- `styles.css`: estilos e responsividade
- `script.js`: interações, animações simples e popup do formulário

## Como publicar no GitHub Pages

1. Crie um repositório no GitHub.
2. Envie estes arquivos para a raiz do repositório.
3. Vá em **Settings > Pages**.
4. Em **Build and deployment**, escolha **Deploy from a branch**.
5. Selecione a branch principal e a pasta `/root`.
6. Salve e aguarde a publicação.

## Ajustes recomendados antes de produção

- Trocar as imagens de fundo por imagens próprias da empresa.
- Conectar o formulário com Formspree, Netlify Forms ou backend próprio.
- Inserir links reais das redes sociais.
- Adicionar favicon e logo oficial em SVG/PNG.

## Observação

O layout foi construído para ser leve, fácil de editar e compatível com hospedagem estática.
