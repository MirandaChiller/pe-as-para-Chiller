# Chiller Peças Site Institucional

Versão estática pronta para GitHub Pages.

## O que foi ajustado
- Logo oficial adicionada por URL externa
- CTA principal apontando para o WhatsApp de orçamento
- Paleta predominante em azul claro com destaques em laranja
- Sem uso de degradê como elemento principal
- Vitrine priorizando categorias mais atraentes da home atual
- Formulário que monta mensagem pronta e abre o WhatsApp

## Publicação no GitHub Pages
1. Crie um repositório no GitHub
2. Envie os arquivos `index.html`, `styles.css`, `script.js` e `README.md`
3. Vá em **Settings > Pages**
4. Em **Build and deployment**, selecione **Deploy from a branch**
5. Escolha a branch principal e a pasta `/root`
6. Salve e aguarde a URL pública

## Observação
A logo está sendo carregada de uma URL externa. Se quiser independência total, baixe a imagem e coloque o arquivo dentro do próprio repositório.
